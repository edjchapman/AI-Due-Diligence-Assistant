AI Due Diligence Assistant

Reads a company’s filings and board minutes, runs four due-diligence checks, and cites the exact source behind every verdict.

keyless demo mode

filings + minutes→ chunk + embed (pgvector)→ LangGraph agent→ cited report→ eval gate in CI

Run a due-diligence report

Pick a company — a stateful LangGraph agent runs four checks (revenue concentration · related-party · going concern · auditor change), each returning a cited verdict corroborated by structured extraction from the filing PDF.
